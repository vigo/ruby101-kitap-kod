## Syntax (Söz Dizimi)

```ruby
puts "Merhaba" if a > 5
```


```ruby
def greet_user(user_name)
  puts "Merhaba #{user_name}"
end

greet_user("Uğur") # Merhaba Uğur
```


```ruby
def greet_user user_name
  puts "Merhaba #{user_name}"
end

greet_user "Uğur" # Merhaba Uğur
```
